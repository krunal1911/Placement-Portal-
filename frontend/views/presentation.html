EXECUTIVE SUMMARY
Problem Statement & Purpose

❌ The Problem

  • Students use fragmented platforms for aptitude tests, technical practice, and resume building.
  • PDF resumes fail Applicant Tracking Systems (ATS) due to unoptimized formatting and missing keywords.
  • Remote online assessments suffer from cheating risks (tab switching, copy-pasting, external help).
  • Placement officers lack real-time visibility into department-wise recruitment pipelines.

✅ The Solution

  • Unified portal bridging Students, Placement Officers, and Assessment Coordinators.
  • Automated 1-step PDF Resume upload with instant 0-100% ATS Readability Scoring.
  • Proctoring suite with live webcam violation snapshots and PDF audit logs.
  • Real-time placement dashboards with department bar charts and candidate status Doughnut charts.

Frontend Interface HTML5, Vanilla CSS3, JavaScript (ES6+), Chart.js Glassmorphic UI, responsive breakpoints (<480px to 4K), theme engine.

Anti-Cheating & Proctoring Suite

🔒 Active Exam Link Control

Tokenized exam links with dynamic time extension (+15m, +30m) and emergency coordinator shutdown triggers.
