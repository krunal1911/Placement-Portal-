📋 Formatting Instructions & Guidelines:

  • Excel (.xlsx / .xls): Required headers: Question, Option A, Option B, Option C, Option D, Answer. Optional headers: Explanation, Topic, Difficulty.
  • PDF (.pdf): Ensure questions follow a standard exam paper layout (e.g., Question text, A) ... B) ... C) ... D) ... Answer: A).
  • Automatic Duplicate Guard: Questions with matching text will be skipped automatically to prevent duplicate questions.
